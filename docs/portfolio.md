## Technical Interview Explanations

### Question 1: "Tell me about a complex project you worked on"
**Answer**: "I built DataNarrate, an AI-powered analytics platform. The biggest challenge was implementing the natural language to SQL system. I used a hybrid approach - rule-based for common queries, and OpenRouter's LLM for complex questions. I added schema-aware prompting, SQL validation, and fallback to Ollama for offline use. For the frontend, I built a smart chart engine that automatically detects column types and selects the best visualization. Performance optimization with Redis caching got queries from 10+ seconds to under 1 second."

### Question 2: "How do you approach system design?"
**Answer**: "For DataNarrate, I followed a layered architecture:
1. Frontend: React + TypeScript for type safety
2. API Layer: FastAPI with Pydantic validation
3. Service Layer: Business logic for SQL generation
4. AI Layer: OpenRouter + Ollama for LLM calls
5. Data Layer: PostgreSQL + Redis cache
I used Docker for containerization and deployed with Vercel/Railway for scalability."
