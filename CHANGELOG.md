# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2025-06-20

### Added
- ✨ Initial release
- 🤖 Natural Language to SQL generation with AI
- 📊 Automatic chart visualization
- 💾 Query history & saved queries
- 📄 Export to CSV, Excel, PDF
- 🔒 SQL validation & security
- ⚡ Redis caching
- 🐳 Docker & Docker Compose support
- 📈 Dashboard with KPIs
- 💡 AI-powered insight generation
- 🎨 Beautiful responsive UI

### Tech Stack
- **Frontend**: React 18, TypeScript, Vite, Tailwind CSS, Chart.js
- **Backend**: FastAPI, Python 3.12, SQLAlchemy, PostgreSQL
- **AI**: OpenRouter, Ollama (fallback)
- **Cache**: Redis
- **DevOps**: Docker, Vercel, Railway, Neon
